## About This Project
 
This repository documents my work throughout the IDX Exchange Data Analyst Internship, a 12-week program focused on analyzing real MLS (Multiple Listing Service) transaction data, engineering key housing market metrics, and building interactive Tableau dashboards to communicate market insights.
 
The program's mission is to teach how real estate analytics datasets are produced, cleaned, and transformed into actionable housing market intelligence using industry-standard tools.
 
---
 
## Tools & Data
 
| Category | Details |
|---|---|
| **Languages/Tools** | Python (Pandas), Tableau Desktop |
| **Data Source** | CoreLogic Trestle API via IDX Exchange pipeline |
| **Dataset Types** | Monthly MLS Listing & Sold transaction CSV files |
| **External Data** | FRED (St. Louis Fed) 30-Year Fixed Mortgage Rate series |
 
---
 
## Program Phases
 
The internship follows a progressive pipeline, with each phase building on the last:
 
1. **Data Cleaning** – Preparing raw data for reliable analysis
2. **Market Analytics** – Engineering key housing market metrics
3. **Competitive Intelligence** – Identifying top agents and brokerages
4. **Dashboard Development** – Building interactive Tableau dashboards
5. **Market Insights** – Communicating findings through reports and presentations
---
 
## Weekly Breakdown
 
| Week(s) | Focus | Key Output |
|---|---|---|
| **0** | MLS Data Pipeline Orientation – learned how monthly datasets are pulled from the CoreLogic Trestle API | Reviewed Trestle Property Metadata |
| **1** | Monthly Dataset Aggregation – concatenated Jan 2024–present listing/sold files | Combined Residential-filtered datasets |
| **2–3** | Dataset Structuring & Validation, plus FRED mortgage rate enrichment | Cleaned datasets merged with 30-yr mortgage rate |
| **4–5** | Data Cleaning – datetime conversion, invalid value handling, date/geo consistency flags | Analysis-ready cleaned dataset |
| **6** | Feature Engineering – price ratio, PPSF, days on market, YrMo, listing/contract/close day metrics | Engineered metrics + segmented summaries |
| **7** | Outlier Detection using IQR method (flag-first approach) | Flagged and filtered datasets |
| **8–10** | Tableau Dashboard Development | `market_analysis.twbx`, `competitive_analysis.twbx` |
| **11–12** | Final Presentation & Reporting | Published dashboards, 1-page report, live presentation |
